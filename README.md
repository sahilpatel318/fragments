# Fragments 

## Overview
This project is a Node.js + Express microservice created for the **Cloud Computing Lab 1** assignment.  
It demonstrates initial setup of a backend service with logging, linting, debugging, and development tooling.  

---

## Prerequisites
Make sure you have the following installed:
- [Node.js (LTS)](https://nodejs.org/en/)  
- [npm](https://www.npmjs.com/) (comes with Node.js)  
- [jq](https://stedolan.github.io/jq/) (for pretty-printing JSON responses)  
- GitHub repo cloned with SSH (`git@github.com:avinav456/fragments.git`)  

---

## Installation
Clone the repo and install dependencies:

```bash
git clone git@github.com:avinav456/fragments.git
cd fragments
npm install
